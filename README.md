MERN Authentication System

This project is a full-stack authentication system built using the MERN stack, which includes MongoDB, Express.js, React, and Node.js. The main purpose of this project is to demonstrate how user authentication works in a modern web application.

The application allows users to create an account, log in securely, and manage their authentication data. The frontend is developed using React, which provides the user interface for signup and login forms. These forms send user data to the backend server using API requests.

The backend is built with Node.js and Express.js, which handle authentication logic, process requests, and interact with the database. User data is stored in MongoDB. When a user logs in successfully, the system generates a token using JSON Web Token to verify the user's identity.

This project demonstrates the integration of frontend and backend technologies, REST API communication, and secure authentication practices using the MERN stack.
